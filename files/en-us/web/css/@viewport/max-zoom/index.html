{{CSSRef}}

The max-zoom CSS descriptor sets the maximum zoom factor of a document defined by the {{cssxref("@viewport")}} at-rule. The browser will not zoom in any further than this, whether automatically or at the user's request.

A zoom factor of 1.0 or 100% corresponds to no zooming. Larger values are zoomed in. Smaller values are zoomed out.

Syntax

/* Keyword value */
max-zoom: auto;

/* <number> values */
max-zoom: 0.8;
max-zoom: 2.0;

/* <percentage> value */
max-zoom: 150%;

Values

auto
The {{glossary("user agent")}} will set the document's upper zoom factor limit.
{{cssxref("<number>")}}
A non-negative number limiting the maximum value of the zoom factor.
{{cssxref("<percentage>")}}
A non-negative percentage limiting the maximum value of the zoom factor.
