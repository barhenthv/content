{{APIRef("DOM")}} {{Non-standard_header}}

The Node.baseURIObject property returns the {{Interface("nsIURI")}} representing the node's (typically a document or an element) base URL. It's similar to {{domxref("Node.baseURI")}}, except it returns an nsIURI instead of a string.

This property exists on all nodes (HTML, XUL, SVG, MathML, etc.), but only if the script trying to use it has UniversalXPConnect privileges.

See {{domxref("Node.baseURI")}} for details on what the base URL is.

Syntax

uriObj = node.baseURIObject

Value

An {{Interface("nsIURI")}}. This property is read-only; attempting to write to it will throw an exception. In addition, this property may only be accessed from privileged code.

Specification

Not in any specification.
