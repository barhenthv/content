The styleMap read-only property of the {{domxref("CSSStyleRule")}} interface returns a {{domxref('StylePropertyMap')}} object which provides access to the rule's property-value pairs.

Syntax

var stylePropertyMap = cssStyleRule.styleMap;

Value

A {{domxref('StylePropertyMap')}} object.
