{{APIRef("DOM Events")}}{{SeeCompatTable}}

The getTargetRanges() property of the {{domxref("InputEvent")}} interface returns an array of static ranges that will be affected by a change to the DOM if the input event is not canceled.

Syntax

var staticRanges[] = inputEvent.getTargetRanges()

Parameters

None.

Return value

An array of {{domxref("StaticRange")}} objects.
