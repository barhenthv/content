The NDEFWriter() constructor of the Web NFC API returns a newly constructed {{DOMxRef("NDEFWriter")}} object used to write NDEF messages to compatiable NFC devices, e.g. NDEF tags, within the reader's magnetic induction field.

Syntax

writer = new NDEFWriter();

Parameters

None.

Return value

A new {{DOMxRef("NDEFWriter")}}.
