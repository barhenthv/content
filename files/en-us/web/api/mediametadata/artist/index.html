{{SeeCompatTable}}{{APIRef("Media Session API")}}

The artist property of the {{domxref("MediaMetaData")}} interface returns or sets the name of the artist, group, creator, etc., of the media to be played.

Syntax

var artist = mediaMetadata.artist
mediaMetadata.artist = artist

Value

A {{jsxref("String")}} containing the name of the artist.
