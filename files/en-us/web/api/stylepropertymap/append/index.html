The append() method of the {{domxref("StylePropertyMap")}} interface adds a new CSS declaration to the StylePropertyMap with the given property and value.

Syntax

StylePropertyMap.append(property,value)

Parameters

property
An identifier indicating the stylistic feature (e.g. font, width, background color) to add.
value
The value the given property should have.

Return value

{{jsxref('undefined')}}.
