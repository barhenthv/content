The SVGMeshElement interface provides access to the properties of {{SVGElement("mesh")}} elements.

{{InheritanceDiagram(600, 140)}}

Properties

This interface doesn't implement any specific properties, but inherits properties from its parent interface, {{domxref("SVGGeometryElement")}}, and implements the properties of {{domxref("SVGURIReference")}}.

Methods

This interface doesn't implement any specific methods, but inherits methods from its parent interface, {{domxref("SVGGeometryElement")}}, and implements the methods of {{domxref("SVGURIReference")}}.
