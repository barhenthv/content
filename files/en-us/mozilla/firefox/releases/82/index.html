This article provides information about the changes in Firefox 82 that will affect developers. Firefox 82 is the current Beta version of Firefox, and will ship on October 20, 2020.

Changes for web developers

Developer Tools

No changes

Console

Debugger

Other tools

HTML

  • The color picker used for <input type="color"> is now keyboard accessible on Windows. ({{bug(1526820)}})
  • Enables the allow-downloads flag for the <iframe sandbox> attribute.  ({{bug(1656212)}})

Removals

CSS

  • Support for the new {{CSSxRef("::file-selector-button", "::file-selector-button")}} pseudo-element has been added. This pseudo-element represents a button inside <input type="file"> element. ({{bug(1635675)}}) ({{bug(1662478)}})
  • Improving error recovery for the {{CSSxRef(":is", ":is()")}} and {{CSSxRef(":where", ":where()")}} pseudo-classes. These pseudo-classes should accept a forgiving selextor list, where an invalid selector in the list will not render the entire list invalid. ({{bug(1664718)}})
  • Change appearance: button to only apply to buttons, the button value of  {{CSSxRef("appearance")}} will therefore behave as auto. ({{bug(1662703)}})

Removals

  • Remove -moz-user-disabled ({{bug(1664432)}})

JavaScript

No changes

Removals

HTTP

  • The Content-Disposition header inline directive will now be ignored if the HTML <a> element's download attribute is specified (for same-origin URLs). Note that if specified the Content-Disposition header filename will be used in preference to a filename specified with the download attribute. ({{bug(1658877)}})

Removals

APIs

No changes

DOM

  • Document.execCommand() is no longer supported for nested/recursive calls (returns false). ({{bug(1634262)}})
  • Element.setPointerCapture() throws the NotFoundError exception if the pointer id is invalid (instead of InvalidPointerId) — as defined in the specification.  ({{bug(1662124)}})
